액티비티는 앱의 동작을 제어하고 
xml은 앱의 모양을 제어한다.

1. 안드로이드 스튜디오가 에뮬레이터를 실행하고 AVD를 로드 한 다음 앱을 설치합니다.
2. 앱이 실행되면 MainActivity로 정의한 액티비티객체가 생성된다.
3. 메인액티비티는 activity_main.xml을 레이아웃으로 사용한다고 지정합니다.
	setContentView(R.layout.activity_main);
4. 액티비티는 레이아웃을 화면에 표시하라고 안드로이드에 지시합니다.

안드로이드 런타임(ART : Android Runtime)

R.java
res 하위 폴더는 이름이 정해져 있음
개발자가 임의로 추가 할 수 없다. 또한 서브폴더를 추가하지 않습니다.

리소스 파일명은 대문자 불가
개발자가 res밑에 추가하면 해당 파일명으로 된 int형 변수가 R.java에 자동으로 추가된다.

컴포넌트는 앱 내에서 독립적인 실행 단위이다.

스마트폰 -> layout -> activity -> java
